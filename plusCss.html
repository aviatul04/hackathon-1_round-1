Pulse CSS Framework

A modern, responsive, and customizable CSS framework built from scratch

Get Started Learn More

Introduction

Pulse CSS is a lightweight, flexible CSS framework designed to help developers build modern interfaces quickly while maintaining full control over the design. Unlike other frameworks, Pulse doesn't force you into rigid design patterns but instead provides a solid foundation that can be easily customized.

Built with clean, semantic class names and powered by CSS variables, Pulse makes it easy to create responsive layouts and consistent UIs across all screen sizes.

Pulse CSS Features:
  • Responsive 12-column grid system
  • Comprehensive utility classes for rapid development
  • Modular components that work right out of the box
  • Dark mode support with CSS variables
  • Accessibility-focused defaults
  • Zero dependencies, just vanilla CSS

Installation

Getting started with Pulse CSS is simple. You can either download the CSS file or use a CDN.

Download

Download the complete CSS file and include it in your project: 

/* Add the link to your HTML */
<link rel="stylesheet" href="path/to/pulse.css">

CDN

Alternatively, you can use the CDN: 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/pulse-css@1.0.0/dist/pulse.min.css">

Basic Template

Here's a basic HTML template to get you started: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Pulse CSS Project</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/pulse-css@1.0.0/dist/pulse.min.css">
</head>
<body>
    <div class="container">
        <h1>Hello, Pulse CSS!</h1>
        <p>Start building your project with Pulse CSS.</p>
    </div>
</body>
</html>

Design Principles

Pulse CSS is built on several core design principles that guide its development:

Responsive First

Every component and utility is designed to work seamlessly across all screen sizes, from mobile to desktop.

Modular & Composable

Components and utilities are designed to work together, allowing you to build complex interfaces from simple building blocks.

Customizable

Built around CSS variables, Pulse makes it easy to customize the framework to match your design system.

Accessible

Components are designed with accessibility in mind, providing a solid foundation for building inclusive interfaces.

Responsive Design

Pulse CSS uses a mobile-first approach to responsive design. All components and utilities are designed to work well on mobile devices first, then adapt to larger screens using media queries.

Breakpoints

Pulse CSS includes five breakpoints, which are used throughout the framework:

Breakpoint Class Prefix Screen Width
Extra Small (default) - (no prefix) Less than 640px
Small sm- 640px and up
Medium md- 768px and up
Large lg- 1024px and up
Extra Large xl- 1280px and up
2X Large 2xl- 1536px and up

Using Responsive Utilities

Many utility classes in Pulse CSS can be applied conditionally at different breakpoints by adding the appropriate breakpoint prefix.

Example: Responsive Column Widths

<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        This column is full width on mobile, half width on tablets, 
        and one-third width on desktops.
    </div>
</div>
